Overview

The Healthcare Patient Analysis System is a data analysis project designed to study patient health data using synthetic datasets. It helps identify health patterns, classify patients based on risk levels, and support better healthcare decision-making.

This project uses Exploratory Data Analysis (EDA) to analyze various health parameters such as blood pressure, sugar level, cholesterol, and heart rate.

🎯 Objectives
Generate synthetic healthcare data
Analyze patient health parameters
Identify high-risk patients
Classify patients into risk categories (Low, Medium, High)
Understand relationships between health factors
Provide insights for early detection and prevention
📂 Dataset Description

The dataset is synthetically generated and includes the following fields:

Column Name	Description
Patient_ID	Unique ID
Age	20–80 years
Gender	Male/Female
Blood_Pressure	80–180
Sugar_Level	70–200
Cholesterol	150–300
Heart_Rate	60–120
Risk	Low / Medium / High
